## apps/

Monorepo home for ConstructingOne applications. Each subdirectory is an
independent app, merged in from a prior standalone repository via
`git subtree` so commit history is preserved.

| Path                          | Source repo                                  |
| ----------------------------- | -------------------------------------------- |
| `apps/ar-reports`             | `cdo45/AR-Reports`                           |
| `apps/core-metrics`           | `cdo45/core-construction-metrics`            |
| `apps/subcontractor-portal`   | `cdo45/subcontractor-billing-portal`         |
| `apps/fixed-assets`           | `cdo45/fixed-assets-tracker`                 |
| `apps/wip-report`             | `cdo45/WIP-Report`                           |
| `apps/ar-billing`             | `cdo45/Vance-AR-Billing`                     |
